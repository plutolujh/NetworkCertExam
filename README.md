# 网络规划师考试学习系统

基于Flask的Web应用，用于软考高级**网络规划师**考试备考学习。

## 考试信息

- **考试时间**：每年11月第二个星期六（2026年11月14日）
- **考试科目**：
  - 综合知识（75道选择题，75分）
  - 案例分析（3-5道问答题）
  - 论文（1道写作题）

## 题库统计

- **题目总数**：757道
- **涵盖年份**：2009-2024年综合知识真题
- **题型**：单选题、多选题、判断题

## 功能特性

### 题库系统
- 支持多种题型：单选题、多选题、判断题、案例分析、论文
- 题目筛选：按类型、分类、难度筛选
- JSON格式批量导入/导出

### 学习功能
- **随机练习**：从题库中随机抽取题目
- **错题本**：自动记录答错题目，强化复习
- **间隔复习**：基于SM-2算法的智能复习计划
- **模拟考试**：75道选择题，模拟真实考试环境

### 知识点管理
- Markdown格式笔记支持
- 分类管理：计算机网络基础、网络规划与设计、网络工程实施、网络运维管理、新一代技术

### 统计功能
- 学习进度可视化
- 正确率统计
- 掌握程度分析（新题、学习中、已掌握）

## 技术栈

- **后端**：Flask + SQLAlchemy
- **数据库**：SQLite
- **前端**：Bootstrap 5 + Chart.js
- **间隔算法**：SM-2

## 题库JSON格式

```json
[
  {
    "type": "single",
    "content": "题目内容",
    "options": ["A. 选项1", "B. 选项2", "C. 选项3", "D. 选项4"],
    "answer": "0",
    "explanation": "解析内容",
    "difficulty": 1,
    "category": "分类名称",
    "tags": "标签1,标签2"
  }
]
```

### 字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| type | 是 | 题型：`single`(单选)、`multi`(多选)、`judge`(判断)、`case`(案例)、`essay`(论文) |
| content | 是 | 题目内容 |
| options | 否 | 选项数组（选择题需要） |
| answer | 是 | 答案（选择题用索引字符串，如"0"表示A） |
| explanation | 否 | 解析内容 |
| difficulty | 否 | 难度1-5，默认1 |
| category | 否 | 分类名称 |
| tags | 否 | 标签（逗号分隔） |

## SM-2 算法说明

间隔重复算法根据用户评分调整复习间隔：

| 评分 | 说明 | 间隔 |
|------|------|------|
| 0-2 | 回答错误 | 重置为1天 |
| 3 | 回答正确但困难 | 保持不变 |
| 4 | 回答正确且一般 | ×1.2 |
| 5 | 回答正确且轻松 | ×1.3 |

## 常见问题

**Q: 如何添加更多题目？**
A: 准备JSON格式文件，访问 /import 页面导入。

**Q: 如何创建知识点笔记？**
A: 访问 /notes，点击"新建知识点"，使用Markdown编辑。

**Q: 模拟考试有多少道题？**
A: 默认75道选择题（综合知识科目）。

**Q: 数据存储在哪里？**
A: SQLite数据库位于 data/exam.db，题库可导出为JSON备份。
