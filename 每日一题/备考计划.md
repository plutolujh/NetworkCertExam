## 五大学习阶段

### 阶段一：案例分析突破（6月）

**目标：案例分析 12+/15分**

| 周次 | 内容 |
|------|------|
| 第1周 | 第一题：网络系统规划设计与优化（SDN、无线、负载均衡） |
| 第2周 | 第二题：网络安全防护方案设计（防火墙、IDS、态势感知、ACL） |
| 第3周 | 第三题：传输网络规划与设计（SDH/MSTP/OTN/DWDM、环形保护） |
| 第4周 | 第四题：网络存储与计算平台设计（DAS/NAS/SAN、分布式存储） |
| 第5周 | 第五题：网络故障诊断与排除（故障分析工具、TCP重传） |
| 第6周 | 薄弱点强化 + 2015-2020真题补做 |

**每日任务：**
- 做一道案例分析题（限时30分钟）
- 对答案，评估得分
- 总结答题套路到"每日一题/案例分析/答题框架.md"

---

### 阶段二：综合知识扫盲（7月-8月）

**目标：综合知识 50+/75分**

| 周次 | 内容 |
|------|------|
| 第1-2周 | 计算机基础（CSMA/CD、PPP、HDLC、OSI七层） |
| 第3-4周 | 网络协议（ARP、DNS、TCP/UDP、三次握手） |
| 第5-6周 | 设备与配置（VLAN、STP、OSPF、ACL） |
| 第7-8周 | 规划与设计（需求分析、CIDR、QoS、双核心） |

**每日任务：**
- 按知识点刷题30道
- 标记"掌握/模糊/不会"
- 周末复习本周错题

---

### 阶段三：论文写作入门（8月）

**目标：论文 1000字以上不跑题**

**每周任务：**
- 背诵3-5篇优秀范文摘要段
- 按模板写一篇论文
- 使用评分标准自评

**论文高频方向：**
1. 网络安全防护体系建设
2. SDN在企业网络中的应用
3. 无线网络规划与部署
4. 数据中心网络架构设计
5. 网络运维管理平台建设

---

### 阶段四：综合强化 + 论文冲刺（9月-10月）

**目标：综合55+，案例14+，论文45+**

- 专项强化薄弱知识点
- 每周2篇论文练习
- 限时模拟真题

---

### 阶段五：模拟考试（11月考前2周）

- 每周末限时模拟
- 全面复习错题本
- 论文模板背熟
